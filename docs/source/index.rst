Install
------------------------------------------------------------------------------

``s3pathlib`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install s3pathlib

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade s3pathlib
